# ci-usage

GitHub Actions の使用量を確認し、月次レポートを表示する。

## 使用方法

```
/ci-usage
```

## 実装

このスキルが実行されると、以下のコマンドを実行してレポートを取得する：

```bash
npm run ci:usage
```

## 出力内容

- **集計期間**: 今月の開始日から現在まで
- **実行統計**: 総実行回数、成功回数、失敗回数
- **使用時間**: 今月の使用分数、無料枠、残り枠、使用率
- **ステータス**: 🟢正常 / 🟡注意 / 🔴警告

## 警告基準

- 🟢 **1,500分未満**: 正常
- 🟡 **1,500-1,800分**: 注意喚起
- 🔴 **1,800分以上**: 警告 + 対策オプション提示

## レポート後の対応

使用量が警告レベル（1,800分以上）に達している場合：
1. `.claude/rules/ci-cost-management.md` を参照
2. 対策オプション（Level 1-3）をユーザーに提示
3. ユーザーの承認を得てから設定変更を実施

## 依存

- `gh` CLI（GitHub CLI）
- `jq`（JSON処理）
- `bash`
